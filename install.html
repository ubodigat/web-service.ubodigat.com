Installation & Nutzung

Wähle deine Umgebung – die Anleitung passt sich automatisch an.

1️⃣ Projektstruktur vorbereiten

Das Projekt wird als ZIP-Paket bereitgestellt und ist modular aufgebaut. Die wichtigsten Dateien sind bereits an den vorgesehenen Stellen platziert:

  • install/setup.php – browserbasiertes Abschluss-Setup
  • sql/struktur.sql – initiale Datenbankstruktur
  • config.sample.php – Konfigurationsvorlage (jeweils vorhanden in projekt/ und filemanager/)

2️⃣ Öffentliche Projektseite (GitHub Pages)

Die Projektwebseite wird über GitHub Pages veröffentlicht und dient ausschließlich zur Dokumentation und zum Bereitstellen der Installationsdateien.

https://web-service.ubodigat.com Klicken zum Kopieren

3️⃣ Installation auf dem Zielserver starten

Auf dem Zielsystem (Debian / Ubuntu) wird die Installation mit einem einzigen Befehl gestartet.

wget https://web-service.ubodigat.com/install/install.sh Klicken zum Kopieren
chmod +x install.sh && sudo ./install.sh Klicken zum Kopieren

Das Skript installiert alle erforderlichen Komponenten und richtet die Projektstruktur automatisch ein.

4️⃣ Browserbasiertes Setup abschließen

Nach Abschluss der Installation wird das Setup im Browser aufgerufen:

http://<server-ip>/install/setup.php Klicken zum Kopieren

Hier werden Administrator-Zugangsdaten sowie Datenbankparameter final konfiguriert.

✅ Installation abgeschlossen

Das System ist nun vollständig eingerichtet und betriebsbereit.

1️⃣ Virtuelle Umgebung erstellen

Erstelle in Proxmox einen Container (CT) oder eine virtuelle Maschine (VM) auf Basis von Debian 12.

  • Netzwerkanbindung per DHCP oder statischer IP
  • SSH-Zugriff aktivieren
  • Ausreichend Speicher und Arbeitsspeicher einplanen

2️⃣ Verbindung zur Umgebung herstellen

ssh root@SERVER-IP Klicken zum Kopieren

3️⃣ Installation durchführen

Die weitere Installation erfolgt identisch zur Standard-Linux-Installation über das bereitgestellte Installationsskript.

1️⃣ Server vorbereiten

Stelle sicher, dass ein aktuelles Debian- oder Ubuntu-System installiert ist und Root- oder sudo-Zugriff besteht.

2️⃣ Installation starten

Lade das Installationsskript direkt von der Projektwebseite herunter und führe es aus.
