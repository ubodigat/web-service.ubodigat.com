Schritt-für-Schritt Anleitung

Webserver Stack installieren

Apache, PHP, MariaDB, Web-Dateimanager, Admin-Panel und 2FA in unter 10 Minuten — ein Befehl genügt.

Ubuntu 22.04 / 24.04 Debian 12 Proxmox LXC VPS / Bare Metal
Plattform:
Schritt 1

Vorbereitung

Stelle sicher, dass du Root-Zugriff auf deinen Ubuntu- oder Debian-Server hast.

  • Ubuntu 22.04 / 24.04 oder Debian 12
  • SSH-Zugriff mit Root oder sudo
  • Aktive Internetverbindung
  • Frisches oder vorher gesichertes System
Schritt 2

Installationsskript laden

Lade das Installationsskript auf deinen Server und setze die Ausführungsrechte.

wget -O install.sh https://web-service.ubodigat.com/install/install.sh Kopieren
chmod +x install.sh Kopieren
Schritt 3

Installation starten

Starte das Installationsskript mit Root-Rechten. Der gesamte Vorgang dauert ca. 5–10 Minuten und ist vollautomatisch.

sudo ./install.sh Kopieren
Vollautomatische Konfiguration

Das Skript konfiguriert MariaDB vollautomatisch — es sind keine manuellen Eingaben erforderlich. Datenbank, Benutzer und Passwort werden im Hintergrund angelegt. Nach Abschluss das Web-Setup unter http://<SERVER-IP>/install/setup.php aufrufen.

Schritt 4

Web-Setup abschließen

Nachdem das Terminal-Skript abgeschlossen ist, öffne die Setup-Seite im Browser um Admin-Zugangsdaten und Einstellungen einzurichten.

http://<deine-server-ip>/install/setup.php Kopieren
Admin-Panel Benutzerverwaltung & Einstellungen bereit
Dateimanager Dateien im Browser verwalten bereit
phpMyAdmin MariaDB Datenbank-Verwaltung bereit

Was nach der Installation verfügbar ist

  • Web-Dateimanager mit Monaco-Editor
  • Chunk-Upload bis 2 GB mit Fortschrittsbalken
  • Bild- & Video-Vorschau im Browser
  • Metadaten-Editor für Mediadateien
  • phpMyAdmin Datenbankzugang
  • Admin-Panel mit 2FA / TOTP
  • Ordnerspezifische Zugriffsrechte (ACL)

  • Ein-Klick-Update auf neue Versionen

Container erstellen

Proxmox LXC Vorbereitung

Erstelle einen Linux Container (LXC) mit Ubuntu 24.04 oder Debian 12 als Template.

  • Mindestens 1 GB RAM zuweisen
  • Mindestens 10 GB Festplattenspeicher
  • Netzwerk-Interface einrichten (DHCP)
  • Option „Nesting" aktivieren (für Docker-Support)
Installation

Container einrichten

Verbinde dich per Proxmox-Konsole oder SSH mit dem Container und führe die Standard-Installation aus.

  • Container starten und Konsole öffnen
  • Als root anmelden
  • Installationsbefehl ausführen (Schritt 2–4 der Linux-Anleitung)
  • Setup im Browser abschließen
Hinweis zu Proxmox

Die Installation in einem Proxmox LXC Container entspricht der Standard-Linux-Installation. Fahre nach der Container-Erstellung mit Schritt 2 der Linux-Anleitung fort.

VPS vorbereiten

Cloud VPS Setup

Funktioniert auf allen gängigen Cloud-Anbietern mit Ubuntu- oder Debian-Image.

  • Hetzner Cloud, DigitalOcean, AWS, Vultr, etc.
  • Ubuntu 22.04 / 24.04 Image wählen
  • SSH-Key beim Erstellen hinterlegen
  • Mindestens 1 vCPU, 1 GB RAM, 20 GB Disk
Firewall prüfen

Netzwerk & Firewall

Stelle sicher, dass die nötigen Ports im Anbieter-Dashboard freigegeben sind.

  • Port 80 (HTTP) öffnen
  • Port 443 (HTTPS) öffnen
  • Port 22 (SSH) öffnen
  • Anbieter-Firewall-Regeln prüfen
Tipp für VPS

Nach dem Erstellen des VPS per SSH verbinden und die Standard-Linux-Installation (Schritte 2–4) ausführen. Für öffentlich erreichbare Server empfehlen wir zusätzlich HTTPS (Let's Encrypt), Fail2ban und regelmäßige Backups.

Einzel-Befehl

Alles in einem Befehl

Laden, Rechte setzen und starten in einer Zeile — kopiere diesen Befehl direkt in dein Terminal.

wget -O install.sh https://web-service.ubodigat.com/install/install.sh && chmod +x install.sh && ./install.sh Kopieren
