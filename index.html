Kurzdefinition

Was ist web-service?

Manuell

Pakete installieren, Datenbank konfigurieren, PHP-Module prüfen, Benutzerrechte bauen, Uploads absichern und Updates pflegen.

web-service

web-service bündelt diese Schritte in einem Open-Source Installationsskript für einen vollständigen Webserver Stack mit Verwaltung im Browser.

Voraussetzungen und Zielgruppe

Für wen ist web-service geeignet?

web-service richtet sich an Betreiber kleiner Webprojekte, Entwickler, Lernumgebungen und Homeserver-Nutzer, die einen Apache-PHP-MariaDB-Stack mit Webverwaltung nachvollziehbar installieren möchten.

Kurzdefinition

web-service ist ein Open-Source Webserver Installationsskript für Ubuntu und Debian. Es richtet Server-Stack, Dateimanager, Admin-Panel, Benutzerverwaltung, 2FA und Updates ein.

Technische Voraussetzungen

Empfohlen wird ein frischer oder gesicherter Ubuntu-/Debian-Server mit SSH-Zugriff, administrativen Rechten und freiem Zugriff auf HTTP/HTTPS.

Geeignete Einsatzbereiche

Das Projekt eignet sich für Testserver, Homeserver, kleine Webprojekte, Lernsysteme und Entwicklungsumgebungen. Produktive Systeme sollten zusätzlich individuell abgesichert werden.

Open Source

Der Quellcode ist öffentlich auf GitHub verfügbar. Die Installation und die statische Projektseite sind ohne externe Frameworks nutzbar.

FAQ

Häufige Fragen zu web-service

Kurze Antworten für Nutzer, Suchmaschinen und KI-Antwortsysteme.

Was ist web-service?

web-service ist ein Open-Source Installationsskript für einen vollständigen Webserver mit Apache, PHP, MariaDB, Web-Dateimanager, Admin-Panel, Benutzerverwaltung, 2FA und Update-System.

Welche Systeme werden unterstützt?

Das Projekt ist für Ubuntu- und Debian-Server ausgelegt. Es sollte auf einem frischen oder vorher gesicherten System ausgeführt werden.

Was installiert das Skript genau?

Das Skript installiert und konfiguriert Apache, PHP, MariaDB, phpMyAdmin, Projektdateien, Web-Dateimanager, Adminbereich, Rollen/Rechte, 2FA und Update-Funktionen.

Ist web-service kostenlos?

Ja. web-service steht unter der Apache License 2.0 und ist vollständig Open Source auf GitHub verfügbar.

Brauche ich Programmierkenntnisse?

Grundkenntnisse im Umgang mit SSH und Linux-Servern sind hilfreich. Die Installation selbst ist auf einen einzigen Befehl reduziert.

Gibt es einen Web-Dateimanager?

Ja. web-service enthält einen browserbasierten Dateimanager mit Upload, Dateiliste, Aktionen, Monaco Editor und Media-Viewer.

Unterstützt web-service 2FA?

Ja. Das Projekt enthält 2FA/TOTP-Funktionen zur Absicherung von Admin-Zugängen.

Kann ich Updates automatisch durchführen?

web-service enthält Versionscheck und One-Click-Updates. Vor Updates sollte trotzdem immer ein Backup vorhanden sein.

Ist das Projekt für Homeserver geeignet?

Ja, besonders für Lern-, Test- und kleine Webserver-Setups. Bei öffentlichem Betrieb sollten Firewall, Backups, HTTPS und Serverhärtung geprüft werden.

Open-Source Webserver Stack für Ubuntu, Debian und Homeserver

web-service hilft beim schnellen Einrichten eines Apache-PHP-MariaDB-Stacks mit browserbasierter Verwaltung. Es ersetzt keine individuelle Sicherheitsprüfung, reduziert aber den manuellen Einrichtungsaufwand deutlich.
